# Politica de Seguridad del Repositorio

## Versiones soportadas
Este proyecto es academico. Se mantiene una sola linea activa en main y dev.

## Reporte de vulnerabilidades
1. No publicar detalles sensibles en issues publicos.
2. Reportar de forma privada al mantenedor del repositorio.
3. Incluir: descripcion, impacto, pasos de reproduccion y mitigacion propuesta.

## Alcance del reporte
1. Scripts Bash y PowerShell.
2. Plantillas Terraform.
3. Documentacion tecnica vinculada a controles de seguridad.
