# 02. Objetivos

## Objetivo General
Disenar, implementar y validar un conjunto de politicas de seguridad informatica aplicadas a una arquitectura cliente-servidor, utilizando un laboratorio virtual simulado para demostrar controles de acceso, autenticacion, proteccion de datos, monitoreo, respaldos, actualizacion de sistemas y respuesta ante incidentes.

## Objetivos Especificos
1. Identificar los activos criticos del entorno cliente-servidor.
2. Analizar riesgos y amenazas comunes que impactan confidencialidad, integridad y disponibilidad.
3. Disenar politicas de seguridad formales, medibles y alineadas a buenas practicas.
4. Implementar controles tecnicos base sobre servidor y clientes del laboratorio.
5. Validar la efectividad de las politicas mediante simulaciones controladas.
6. Documentar resultados, limitaciones y recomendaciones en formato academico.

## Indicadores de Cumplimiento
1. Porcentaje de politicas implementadas respecto al diseno.
2. Numero de simulaciones ejecutadas con evidencia valida.
3. Trazabilidad completa entre riesgos, controles y resultados.
4. Porcentaje de eventos de seguridad detectados y registrados correctamente.
