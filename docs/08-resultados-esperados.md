# 08. Resultados Esperados

## Resultados Tecnicos
1. Endurecimiento basico del servidor Linux aplicado y verificable.
2. Control de acceso por roles funcionando conforme a politica.
3. Comunicaciones administrativas seguras por SSH y/o HTTPS.
4. Proceso de respaldo y restauracion probado con exito.
5. Registros de auditoria disponibles y utiles para investigacion.
6. Procedimiento de respuesta a incidentes documentado.

## Resultados Academicos
1. Coherencia metodologica entre objetivos, riesgos, politicas y pruebas.
2. Evidencia suficiente para sustentar defensa de maestria.
3. Reproducibilidad del laboratorio por terceros.
4. Justificacion tecnica de decisiones de plataforma e implementacion.

## Criterios de Aceptacion
1. Se ejecutan las seis simulaciones con evidencia completa.
2. Se demuestra mitigacion parcial o total de riesgos prioritarios.
3. Se presentan limitaciones y mejoras futuras de forma honesta y sustentada.
