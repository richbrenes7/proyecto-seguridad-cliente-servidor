# 12. Lista de Tareas Operativas

## Fase 1. Planeacion
1. Definir alcance tecnico y academico.
2. Documentar supuestos y restricciones del laboratorio.
3. Definir entregables, evidencia y criterios de exito.
4. Seleccionar plataforma de virtualizacion y herramientas.

## Fase 2. Diseno del Laboratorio
1. Definir topologia y segmentacion de red.
2. Especificar VMs, roles, recursos y direccionamiento.
3. Definir dependencias tecnicas y prerequisitos.

## Fase 3. Infraestructura como Codigo
1. Validar viabilidad de Terraform segun plataforma.
2. Ajustar variables y plantilla base.
3. Ejecutar plan y aplicar aprovisionamiento inicial.
4. Registrar limitaciones del proveedor elegido.

## Fase 4. Configuracion del Entorno
1. Preparar servidor Linux base.
2. Crear usuarios, grupos y permisos.
3. Endurecer SSH y control de sesiones.
4. Configurar firewall y servicios minimos.
5. Habilitar logging y auditoria.
6. Configurar respaldo y restauracion.
7. Aplicar politica de parches.

## Fase 5. Politicas Formales
1. Redactar politicas con objetivo, alcance y lineamientos.
2. Definir roles responsables y frecuencia de revision.
3. Establecer mecanismo de cumplimiento.

## Fase 6. Simulaciones
1. Ejecutar S1 a S6 conforme a procedimiento.
2. Capturar evidencia tecnica de cada prueba.
3. Documentar resultados reales y desviaciones.

## Fase 7. Resultados y Cierre
1. Consolidar matriz final de riesgos mitigados.
2. Elaborar analisis de hallazgos y limitaciones.
3. Emitir recomendaciones para trabajo futuro.
